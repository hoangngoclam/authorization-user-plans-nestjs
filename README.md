## Description

This for demo about Authorization in case user plan
example: 
- user with pro plan will be interact with all api
- user with medium plan will be interact with only some get api

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev
```
